# Docs Archive Index

This directory is the archive index for historical project material.

## Canonical documents stay outside archive

- `README.md` is the repository entrypoint.
- `ROADMAP.md` is the canonical active roadmap.
- `docs/FUNCTION_MAP.md` is the canonical function-tree reference.
- `docs/USER_MANUAL.md` is the canonical user and operator guide.

## Retention buckets

- Legacy `docs/plans` content is now archived under `docs/archive/plans`.
- Legacy `docs/reports` content is now archived under `docs/archive/reports`.
- `docs/archive/plans` contains historical plan snapshots.
- `docs/archive/reports` contains historical implementation and verification reports.
- `docs/archive/ad-hoc` contains one-off historical notes and temporary analysis documents.

## Archival rule

Move historical material here once it is no longer a canonical entry document or an actively
maintained working document. When archiving, leave a short redirect from any old high-traffic
location if readers may still arrive there through existing links.
