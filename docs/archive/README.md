# Documentation Archive

This directory contains historical project material that is useful for audit,
forensics, or context recovery, but is no longer a primary entrypoint for current
work.

## Canonical Documents Stay Outside Archive

- `README.md` remains the project entrypoint.
- `FUNCTION_TREE.md` remains the function status registry and planning
  authority.
- `docs/USER_MANUAL.md` remains the operator manual.
- Current cleanup, audit, and evidence reports remain in `docs/reports/` until
  they are explicitly retired.

## Buckets

- `docs/archive/plans/` contains completed or superseded implementation plans.
- `docs/archive/reports/` contains completed phase, audit, and verification
  reports that no longer describe current work.
- `docs/archive/ad-hoc/` contains one-off analysis notes and short-lived planning
  documents.

Move material here only when it is no longer a canonical entry document, active
implementation plan, or current evidence report.
