命令路径：quantix backtest run

运行一次回测并保存结果

用法

quantix backtest run [OPTIONS] --code <CODE>

参数

名称值说明
--strategySTRATEGY策略名称，当前优先支持 ma_cross [default: ma_cross]
--codeCODE股票代码
--startSTART开始日期 (YYYYMMDD)
--endEND结束日期 (YYYYMMDD)
--capitalCAPITAL初始资金 [default: 100000]
--commission-rateCOMMISSION_RATE手续费率 [default: 0.0003]
--slippage-bpsSLIPPAGE_BPS滑点，单位 bps [default: 10]

--longLONG_PERIODMA 长周期 [default: 20]
--max-positionsMAX_POSITIONS最大持仓数量 [default: 5]
--max-position-ratioMAX_POSITION_RATIO单股最大持仓比例 [default: 0.2]
--risk-free-rateRISK_FREE_RATE无风险利率（年化） [default: 0.03]
--limitLIMIT限制加载 K 线条数 [default: 10000]
